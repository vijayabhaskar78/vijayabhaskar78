# Vijaya Bhaskar — AI Developer Tooling

I build experimental infrastructure for **AI-assisted software engineering** and **autonomous developer workflows**.

## Focus Areas

• AI coding agents 
• Long-context memory systems for LLM workflows 
• Autonomous repository maintenance

## Key Projects

**Repogent**
Autonomous AI agents that manage GitHub repositories by triaging issues, reviewing pull requests, diagnosing CI failures and assisting maintainers.

**SSM**
Structured memory experiments for improving reliability of LLM-driven systems.

**ContextForge**
Tooling for maintaining long-context reasoning in AI agent workflows.

**mini-distributed-search-engine**
Production-style distributed search engine with crawler, indexer and ML ranking pipeline.

## Goal

Build infrastructure that allows **AI systems to act as reliable collaborators in real software engineering workflows.**
